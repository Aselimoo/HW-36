getElementById — быстрый и простой метод для получения элемента по его id.
getElementsByClassName — удобен, если нужно получить все элементы с определенным классом, но возвращает живую коллекцию.
querySelector — универсальный метод, работающий с любыми CSS-селекторами, но возвращающий только первый элемент.

querySelectorAll — аналог querySelector, но возвращает все совпадающие элементы в виде статической коллекции.

DOM (Document Object Model) — это программный интерфейс для HTML и XML документов.

Он представляет документ как дерево объектов, где каждый элемент HTML является объектом.

DOM позволяет языкам программирования, таким как JavaScript, взаимодействовать с документом, изменять его структуру.
